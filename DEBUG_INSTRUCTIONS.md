# 🔍 紧急调试指南

## 请立即执行以下步骤

### 步骤 1: 打开浏览器控制台
1. 打开浏览器（Chrome/Edge/Firefox）
2. 按 F12 键（或右键 → 检查）
3. 点击 "Console" 标签页

### 步骤 2: 清空控制台
点击控制台左上角的 🚫 图标清空所有日志

### 步骤 3: 开始诊断
1. 在页面上点击"开启今日问诊"
2. 上传任意一张图片
3. 点击"开始诊断"

### 步骤 4: 等待并观察
等待 30-70 秒，观察控制台的日志输出

### 步骤 5: 复制所有日志
1. 在控制台中右键
2. 选择"全选"或按 Ctrl+A (Mac: Cmd+A)
3. 复制所有内容
4. 粘贴给我

## 如果页面完全空白

请检查：
1. 浏览器控制台是否有红色错误信息
2. Network 标签页中是否有失败的请求（红色）
3. 页面源代码是否正常加载（右键 → 查看页面源代码）
